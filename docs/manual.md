# XML Parser Service

Сервис для обработки XML-файлов и управления данными в PostgreSQL базе данных.

## Особенности

- Использует Groovy с XmlSlurper для парсинга XML
- Работает с PostgreSQL через JDBC
- Предоставляет консольный интерфейс для взаимодействия
- Следует принципам SOLID, Clean Code, DRY, KISS

## Функциональность

- Чтение XML-файлов по веб-ссылке
- Динамическое создание SQL-запросов для создания таблиц
- Обновление данных в базе данных
- Проверка уникальности столбцов
- Обнаружение изменений структуры таблиц

## Требования

- Java 21
- PostgreSQL
- Docker (для запуска PostgreSQL в контейнере)

## Запуск

1. Запустите PostgreSQL с помощью Docker Compose:
   ```bash
   docker-compose up -d
   ```

2. Соберите и запустите приложение:
   ```bash
   ./gradlew bootRun
   ```

## Использование

После запуска приложения появится интерактивный консольный интерфейс с следующими опциями:

1. Просмотр доступных таблиц
2. Генерация DDL для таблицы
3. Обновление всех таблиц в БД
4. Обновление конкретной таблицы в БД
5. Получение списка столбцов таблицы
6. Проверка уникальности столбца
7. Получение изменений DDL для таблицы
8. Выход из приложения
